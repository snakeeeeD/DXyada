# RibBond

## 概要
コントローラーのスティックを回転させてリボンを巻き取り、
敵やオブジェクトを「飾る」ことでクリアしていく横スクロールアクションゲーム。
「飾る喜びが、より高いクリアの快感を生み出す」というコンセプトのもと、
チーム（プログラマー5名）で制作。

# Ribbond - 個人の担当範囲について

本プロジェクトはチーム制作（プログラマー5名）です。
明確に担当が分かれていたのはプレイヤー処理、リボン処理で、私はプレイヤー処理を担当しました。

担当範囲：プレイヤーキャラクターの操作・移動・リボンを巻き取る操作・ジャンプ処理
（コントローラーのスティック回転を検出し、リボンの巻き取り動作と
ジャンプ挙動に変換する処理を実装）

ステージ等、一部の領域は複数メンバーが共同で触れていたため、
ファイル単位での厳密な切り分けが難しい部分があります。
上記のプレイヤー処理が、私が責任を持って担当した範囲です。
