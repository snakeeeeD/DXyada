# RibBond

## 概要
コントローラーのスティックを回転させてリボンを巻き取り、
敵やオブジェクトを「飾る」ことでクリアしていく横スクロールアクションゲーム。
「飾る喜びが、より高いクリアの快感を生み出す」というコンセプトのもと、
チーム（プログラマー5名）で制作。
